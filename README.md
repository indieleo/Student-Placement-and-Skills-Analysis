# Student Placement and Skills Analysis

This project explores factors influencing student placement outcomes.

## Objectives
- Perform exploratory data analysis
- Identify key placement drivers
- Build a classification model
- Evaluate feature importance

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib / Seaborn

## Key Insights
- Internship experience strongly influenced placement probability
- Technical projects had higher predictive power than CGPA alone
- Built a Random Forest model to predict placement outcomes
